# Challenge Roadmap

Uma plataforma educacional completa com 200 desafios de programação organizados por níveis de dificuldade.

## Sobre o Projecto

O Challenge Roadmap é uma plataforma inteligente que guia desenvolvedores em uma jornada estruturada de aprendizado através de desafios de programação progressivos.

### Características

- **200 Desafios** organizados em 5 níveis de dificuldade
- **Multi-linguagem** - Resolva em Python, JavaScript, Java, C#, C++ ou qualquer linguagem
- **Sistema de Progresso** - Acompanhe seu desenvolvimento
- **Gamificação** - Pontuação e níveis
- **Interface Responsiva** - Funciona em desktop e mobile

## Níveis de Dificuldade

1. **Iniciante** (1-30) - Fundamentos da programação
2. **Intermediário** (31-70) - Algoritmos e estruturas básicas
3. **Avançado** (71-100) - Algoritmos complexos
4. **Expert** (101-150) - Problemas desafiadores
5. **Master** (151-200) - Desafios de nível profissional

## Como Executar

### Pré-requisitos
- Python 3.8+
- pip (gerenciador de pacotes Python)

### Instalação

1. **Clone o repositório**
```bash
git clone https://github.com/seu-usuario/challenge-roadmap.git
cd challenge-roadmap
```

2. **Instale as dependências**
```bash
pip install -r requirements.txt
```

3. **Execute a aplicação**
```bash
python app.py
```

4. **Acesse no navegador**
```
http://localhost:5000
```

## Tecnologias Utilizadas

- **Backend**: Flask (Python)
- **Frontend**: HTML5, CSS3, JavaScript, Bootstrap 5
- **Database**: SQLite
- **Ícones**: Font Awesome

## Funcionalidades do Usuário

- Registro e autenticação

## Como Usar

1. **Crie uma conta** na plataforma
2. **Explore os desafios** por nível de dificuldade
3. **Escolha um desafio** e leia a descrição
4. **Escreva sua solução** na linguagem de sua preferência
5. **Submeta o código** e acompanhe seu progresso
